Where the data comes from

The data that is displayed on the dashboard landing page of this website is aggregated from The Wisconsin Department of Health Services. They have an interactive map using this data, but it doesn't show city-wide just for each census tract. You can find Wisconsin's interactive map here.

All the data is based around census tract numbers which you can find here. Some of these tracts extend beyond the categories listed on the landing page of the website. I've tried my best to keep the census tract numbers for each of the categories to be accurate as possible, but it is possible I messed up somewhere.
